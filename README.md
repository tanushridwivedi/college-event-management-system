# College Event Management System

A clean, responsive web application for managing college events — built with HTML, CSS, and JavaScript.

## Features

- **Dashboard** — Metrics overview, upcoming events, notifications
- **Events Page** — Browse, search, and filter events by category
- **Registrations** — View and manage your registered events
- **Add Event** — Admin form to create new events

## How to Run

1. Extract the zip file
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge)
3. No server or installation needed — it runs fully in the browser!

## Project Structure

```
college_events/
├── index.html     # Main HTML structure
├── style.css      # All styling
├── app.js         # JavaScript logic & data

```

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- Vanilla JavaScript (ES6+)
- Tabler Icons (via CDN)
